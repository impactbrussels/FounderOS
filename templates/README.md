# Founder OS Templates

Fill-in starting points for the artifacts founders actually have to produce. Each one pairs with
a Founder OS skill that explains the *thinking* - the template just gives you the *shape*.

## What's here

| Template | Use it for | Pairs with skill |
|----------|-----------|------------------|
| [`pitch-deck-outline.md`](pitch-deck-outline.md) | A 10-12 slide investor deck, one job per slide | `pitch-deck`, `investor-pipeline` |

| [`prompt-scaffold.md`](prompt-scaffold.md) | The shared prompt skeleton every Founder OS skill is built on | `founderos-prompt-scaffold` |

## How to use the placeholders

Every template uses the Founder OS placeholder glossary. Find the `[BRACKETED]` tokens, replace
each with your real value, then delete the guidance lines.

| Placeholder | Means |
|-------------|-------|
| `[STARTUP_NAME]` | Your company name |
| `[ONE_LINER]` | One sentence anyone understands |
| `[STAGE]` | Idea & Validation / Build / Launch & GTM / Fundraise / Operate & Scale |
| `[ICP]` | Your one specific ideal customer |
| `[PROBLEM]` | The painful job they're failing to get done |
| `[RISKIEST_ASSUMPTION]` | The belief most uncertain *and* most fatal if wrong |
| `[CONSTRAINT]` | Your binding limit (time, money, skills) |
| `[GOAL]` | The concrete outcome you want |
| `[CHANNEL]` | A repeatable path to reach buyers |
| `[METRIC]` | The number you steer by |
| `[TONE]` | The voice you want the output in |

## Two house rules

1. **Every number is real or labelled an estimate.** Never paste in an invented market size or
   financial figure - sources and assumptions belong next to the number.
2. **A filled template is a draft, not a decision.** Pressure-test it with the paired skill (or a
   Founder OS advisor agent) before you act on it.
